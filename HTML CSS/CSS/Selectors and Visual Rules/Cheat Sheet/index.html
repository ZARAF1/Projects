Tag Name Description

Basic Tags
< html > HTML Creates an HTML document
< head > Head Sets off the title & other info that isn't displayed
< body > Body Sets off the visible portion of the document
< title > Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked

Text Tags
< pre > Preformatted Text Creates preformatted text
< h1 > < /h1 > --- < h6 > < /h6 > Headlines 1-6 Creates headlines -- H1=largest, H6=smallest
< b > Bold Creates bold text (should use < strong > instead)
< i > Italic Creates italicized text (should use < em > instead)
< tt > Type Writer Text Creates typewriter-style text
< code > Code Style Used to define source code, usually monospace
< cite > Citation Creates a citation, usually processed in italics
< address > Address Creates address section, usually processed in italics
< em > Emphasis Emphasizes a word (usually processed in italics)
< strong > Strong Emphasizes a word (usually processed in bold)
< font size=? > Font Size Sets size of font - 1 to 7 (should use CSS instead)
